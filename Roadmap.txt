개발자 취업 방향

1. 풀스택 개발자 (Full stack Developer)
	* 프론트엔드, 백엔드 모두 다룰 줄 앎
	* 전체적인 시스템의 구축과 유지보수 담당

2. 프론트엔드 개발자 (Front-end Developer)
	* 웹 어플리케이션의 사용자 인터페이스를 개발
	* HTML, CSS, JavaScript 등을 사용하여 시각적인 부분 구축

3. 백엔드 개발자 (Back-end Developer)
	* 서버 측에서 데이터 처리, 로직 구현, 데이터베이스 관리 등 서버와 데이터 처리를 담당
	* 서버 사이드언어, 데이터베이스 기술

4. 모바일 앱 개발자 (Mobile App Developer)
	* 안드로이드, ios 개발자
	* 각 플랫폼에 맞는 언어와 프레임워크를 사용

5. 데이터베이스 개발자 (Database Developer)
	* 데이터베이스 시스템 설계, 구축, 유지보수
	* SQL 등 

6. 데브옵스 엔지니어 (DevOps Engineer)
	* 개발과 운영을 통합하여 소프트웨어 제공과 배포를 자동화
	* 시스템의 안정성과 성능을 관리
	* 자동화 도구와 프로세스를 사용하여 개발과 운영 사이의 협력을 강화

7. 시스템 개발자 (Systems Developer)
	* 운영체제, 네으퉈크, 서버 구성등 시스템 수준의 개발
	* 하드웨어와 소프트웨어를 통합하여 안정적인 시스템 운영

8. 블록체인 개발자 (Blockchain Developer)
	* 분산 원장 시스템 구축, 스마트 계약등을 개발

9. 인공지능 개발자 (Artificial Intelligence Developer)
	* 기계학습, 딥러닝 등의 인공지능 기술을 활용
	* 대용량의 테이터 처리와 알고리즘 개발

10. 게임 개발자 (Game Developer)
	* 게임 개발자는 게임 소프트웨어를 개발하는 역할
	* 게임 엔진과 프로그래밍 언어를 사용하여 게임의 구성 요소를 개발

11. 보안 엔지니어 (Security Engineer)
	* 시스템과 앱의 보안 취약점을 분석, 보호
	* 해킹, 악성 코드 방지와 보안 정책 설정에 대한 전문적인 지식과 기술
